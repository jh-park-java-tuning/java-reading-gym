## 앱에서 모호한 부분 밝히기

이 장의 주요 내용
- 코드 조사 기법
- 자바 앱을 이해하기 위한 코드 조사 기법

개발자는 다양한 업무를 맡는 사람이다. 그중 대부분은 개발자 자신이 만지작거리는 코드를 어떻게 이해할것인가.<br>
하는 문제와 밀접하게 연관되어있다. 그들은 문제를 해결하고 새로운 기술을 터득하고자 상당한 시간을 <br>
코드에 할애한다. 생산성을 높이기 위해 효율적인 조사 기법을 연마할 필요가 있다.<br>
코드를 효율적으로 이해하는 방법이 이 책의 중심 주제이다.

> 일반적으로 소프트웨어 개발보다, 작동 방식을 이해하는데 더 많은 시간을 쓴다.

- 소프트웨어 개발자는 조사 기법을 디버깅이라고 뭉뚱그리지만,
<br>
- 디버그는 여러 도구중 하나에 불과하다.
