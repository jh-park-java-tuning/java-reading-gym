원래는 **자바 성능 튜닝 이야기**를 읽고 있었다.

성능도 중요하지만, 그 전에 코드의 흐름과 의도를 빠르게 파악하는 일이 먼저라고 판단했다.  
그래서 우선 **자바 잘 읽는 법**을 먼저 끝내고, 이후에 **자바 성능 튜닝 이야기**로 다시 돌아갈 예정이다.
